3.1 syntax

3.1.1 case-sensitivity

ECMAScript中的一切都是case-sensitive

var test = 12,
Test = 15;
console.log(test)   //12
console.log(Test)   //15

3.1.2 identifier

第一个字符必须是字母, underscore(_), or dollar sign($)

其他字符可以是字母, underscore, dollar sign or number

ECMAScript中的identifier采用camel case(驼峰大小写格式), 这是一种best practice

 *  也叫作多行注释

3.4 data type

ECMAScript中有5种简单data type:Undefined, Null, Boolean, Number, String.还有一种复杂data type:Object

3.4.1 the typeof operator

typeof是一个operator, 不是function,用来检测给定variable的data type,会返回一个字符串

var message;
typeof message      //"undefined"

typeof null         //"object"
typeof true         //"boolean"
typeof 12           //"number"
typeof "hello"      //"string"

3.4.2 the undefined type

Undefined类型只有一个value: undefined.在声明一个variable但未对其初始化时,这个variable的value就是undefined

var message;
typeof message      //"undefined"

3.4.3 the null type

Null类型只有一个value: null.通常表示空指针,用来初始化一个准备保存对象的variable

3.4.4 the boolean type

字面值

true和false.这2个value是case-sensitive

Boolean类型虽然只有2个字面value, 但是ECMAScript中所有类型的值都有与这2个value等价的值

类型转换

Boolean(value: any)
参数任何data type的值
返回值Boolean值
功能将一个值转换为其对应的Boolean值

具体转换规则如下表:

data typeto trueto false
Booleantruefalse
Number任何非0数字0 and NaN
String任何非空字符串""
Object任何对象null
Undefinedundefined

在流控语句比如if中, Boolean()会自动执行, 具体如下:

var message = "hello";
if(message){
console.log("value is true");   //此时会自动执行Boolean(message),然后返回true并执行对应的语句
}

3.4.5 the Number type

数值字面量

  • 十进制(decimal)
  • 八进制(octal), 第一位数字必须是0, 然后是八进制数字(0-7)
  • 十六进制(hexadecimal), 前2位字符必须是0x, 然后是十六进制数字(0-9, a-f)

在进行算术计算时所有的八进制和十六进制表示的数值最终都被转换为十进制数值

var n1 = 10,
n2 = 010,
n3 = 0x10;
console.log(n1);    //输出10
console.log(n2);    //输出8
console.log(n3);    //输出16

浮点数

var n1 = 1.1,
    n2 = 3.124e5;

0.1 + 0.2 = 0.30000000000000004, 而不是0.3

数值范围

Number.MAX_VALUE, ESMAScript能够表示的最大数值; Number.MIN_VALUE, ESMAScript能够表示的最小数值

Infinity(无穷), 如果某一次计算结果不在最大值和最小值之间, 那么这个结果会被自动转换为这个特殊的值

NaN(Not a Number)

用来表示一个本来要返回数值的操作数但未返回数值的情况

isNaN(value: any)
参数任何data type的值
返回值Boolean值
功能确定参数是否"不是数值"

类型转换

Number(value: any)
参数任何data type的值
返回值Number值 or NaN
功能把参数转换为其对应的数值

具体转换规则如下:

  • 如果是Boolean值, true和false分别被转换为1和0
  • 如果是null, 返回0
  • 如果是undefined, 返回NaN
  • 如果是字符串, 则:
    1. 如果字符串中只包含数字或有效的浮点数, 则将其转换为数字
    2. 如果是空, 则返回0
    3. 如果不是数字, 则返回NaN
  • 如果是对象, 则:
    1. 先调用其valueOf()方法, 然后按照上面的规则转换valueOf()方法的返回值;
    2. 如果第一步中转换的结果是NaN, 则调用其toString()方法, 然后按照上面的规则转换toString()方法的返回值;
parseInt(s: string, [radix: integer])
参数s, string, 要转换的字符串
radix, number, 可选
返回值integer or NaN
功能把字符串转换为其对应的整数
parseFloat(s: string)
参数任何data type的值
返回值Number值 or NaN
功能把字符串转换为其对应的浮点数

3.5 operator

在ECMAScript中operator最变态的地方在于, 可以用于所有类型的值

3.5.1 unary operator

increment/decrement

var age = 12;
var anotherAge = ++age + 5;
console.log(age);       //13
console.log(anotherAge);        //18

前置递增会让操作数的结果加1, 且加1发生在操作数被计算之前; 前置递减与之类似

var age = 12;
var anotherAge = age++ + 5;
console.log(age);       //13
console.log(anotherAge);        //17

后置递增会让操作数的结果加1, 且加1发生在操作数被计算之后; 后置递减与之类似

unary plus and minus

在对非数值应用一元加/减操作符时, 会通过Number()把非数值转换为数值

console.log(+true);    //1
var s ="10";
var s1 = "hello";
console.log(s);     //10
console.log(+s1);   //NaN
console.log(+new Object());     //NaN
console.log(+new Date());   //1575612685962, 代表当前日期和时间的毫秒数

3.5.2 boolean operator

logical not

由(!)表示, 适用于所有类型的值, 返回一个Boolean值

操作过程是通过Boolean()先把操作数转换为一个Boolean值, 然后再对其求反

logical and

由(&&)表示, 适用于所有类型的值, 返回一个 Boolean or 其他类型的值

操作过程如下:

  1. 求第一个操作数的Boolean值, 如果不是Boolean类型则通过Boolean()转换
  2. 如果第一个操作数的Boolean值为false, 则直接返回第一个操作数;否则返回第二个操作数

logical and的经典用法: window.loader && loader.done(); 作用是在调用对象的方法之前确保对象已经存在

logical or

由(||)表示, 适用于所有类型的值, 返回一个 Boolean or 其他类型的值

操作过程如下:

  1. 求第一个操作数的Boolean值, 如果不是Boolean类型则通过Boolean()转换
  2. 如果第一个操作数的Boolean值为true, 则直接返回第一个操作数;否则返回第二个操作数

logical or的经典用法: var o = preferredObject || backupObject; 作用是确保为变量赋值一个有效值

解析如下:

  • preferredObject中包含优先赋给变量o的值
  • backupObject负责在preferredObject中不包含有效值的情况下的提供备选值
  • 如果preferredObject的值不是null or undefined, 则它的值将被赋值给变量o; 否则将backupObject的值赋值给变量o
  • 通常情况下bakcupObject都是{}(即已知的有效值)

3.5.3 multiplicative operator

乘(*),除(/)和取余(%)这三个operator, 在操作过程中如果有一个操作数不是数值, 则在后台中调用Number()将其转换为数值

3.5.4 addtive operator

  • 加法(+):
    1. 如果有一个操作数不是数值, 则会把这个操作数转换为字符串
    2. 如果有一个操作数是字符串, 则会把另一个操作数转换为字符串, 然后返回这2个字符串的拼接而成的字符串
  • 减法(-): 如果有一个操作数不是数值, 则调用Number()把这个操作数转换为数值

3.5.5 relational operator

大于(>), 小于(<), 大于等于(>=)和小于等于(<=)这四个operator, 在操作过程中遵循如下规则:

  • 如果2个操作数都是字符串, 则比较这2个字符串首字母对应的编码值
  • 如果有一个操作数不是数值, 则在后台中调用Number()将其转换为数值

3.7.1 argument

arguments对象

是一个类数组对象, 保存着function中传入的所有参数, 可以通过[]语法来访问每一个参数, 通过length属性来确定传入参数的个数(这个length的值是由传入参数的个数决定的,不是由定义function时的命名参数决定的)

这个属性只能在函数内容访问

function method1(){
    console.log(arguments.length);
    console.log(arguments[0]);
}
method1();          //0, undefined
method1("lzw");     //1, lzw
method1("lzw", 30); //2, lzw

3.7.2 no overloading

不存在overloading, 如果前后存在2个名字相同的function, 则该名字只属于后面定义的function
