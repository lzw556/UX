1.1 a short history

javascript诞生于1995年,由netscape公司的Brendan Rich开发.当时大多数用户的网速比较慢,对于一个简单的表单验证,从用户提交数据到服务器返回结果的整个过程都非常耗时,这时候就迫切需要一种客户端语言来处理这些简单的表单验证,正是在这种背景下,netscape公司开发了javascript

1.2 javascript implementation

1.2.1 ECMAScript

由ECMA这个组织建立的ECMA-262标准定义了ECMAScript这门语言,主要内容包括:语法,数据类型,关键字和保留字,操作符等,ECMAScript和web浏览器无关

ECMAScript版本:1, 2, 3, 3.1, 5

1.2.2 the document object model

DOM标准由W3C组织建立
