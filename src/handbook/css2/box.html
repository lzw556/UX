CSS box model描述了一个rectangular box(矩形框),这个框由文档树中的元素生成,根据visual formatting model(视觉格式化模型)进行布局.

8.1 Box dimensions

每一个矩形框都有一个content area,和周围可选的padding area,border area和margin area

content area
content area是一个矩形,其尺寸由box的width和height指定.
其边界叫作content edge or inner edge.
4个content edge组成的矩形叫作box的content box(也就是content area组成的矩形)
padding area
padding area是一个矩形然后去掉中间的content area部分,其尺寸由box的padding指定.
其外边界叫作padding edge(内边界就是content edge)
4个padding edge组成的矩形叫作box的padding box(实际是由content area + padding area所组成的矩形)
border area
border area是一个矩形然后去掉中间的content area和padding area部分,其尺寸由box的border-width指定.
其外边界叫作border edge(内边界就是padding edge)
4个border edge组成的矩形叫作box的border box(实际是由content area + padding area + border area所组成的矩形)
margin area
margin area是一个矩形然后去掉中间的content area和padding area和border area部分,其尺寸由box的margin指定.
其外边界叫作margin edge or outer edge(内边界就是border edge)
4个margin edge组成的矩形叫作box的margin box(实际是由content area + padding area + border area + margin area所组成的矩形)

content area,padding area和border area的背景样式由background指定.margin area的背景总是透明的

8.2.1 Collapsing margins

2个or多个box(可能但不一定是兄弟)中相邻的margin会被结合形成一个margin.margin按这种方式结合叫作collapse,产生的margin叫作collapsed margin

相邻的margin会collapse,除了:

  • horizontal margin不会collapse
  • 根元素生成的box的margin不会collapse
  • 如果带clearance的元素的margin-top和margin-bottom相邻,该元素的margin会和后面紧挨着的兄弟元素的相邻的margin collapse,但这个产生的collapsed margin不会再和父级块的margin-bottom collapse
  • 如果一个box的'height'属性的computed value是'auto'且'min-height'属性的computed value不是0,这个box的margin-top和其最后一个in-flow子元素的margin-bottom collapse,那么这个子元素的margin-bottom不会和父元素的margin-bottom collapse

2个margin相邻的满足条件:

  • 都属于in-flow block-level boxes,这些boxes处于同一个block formatting context中
  • 2个margin之间不能存在line boxes,clearance,padding,border
  • 都属于vertically-adjacent的box edges.即构成下列某一对:
    • box的margin-top和这个box内第一个in-flow子box的margin-top
    • box的margin-bottom和这个box后面紧挨着的in-flow兄弟box的margin-top
    • 最后一个in-flow子box的margin-bottom和其父box的margin-bottom,如果这个父box的height的computed value为auto
    • 一个box的margin-top和margin-bottom,要求这个box没有建立新的block formatting context,并且'min-height'的computed value为0,'height'的computed value为0 or auto,并且没有in-flow子box

注意:上面的规则表明:

  • floated box和其他任何box之间的margin不会collapse,甚至浮动box和其in-flow子box之间也不会
  • 会建立新的bfc的元素(像float元素或'overflow'值不是'visible'的元素)的margin不会和他们的in-flow子元素的margin进行collapse
  • 绝对定位box之间的margin不会collapse
  • inline-block box之间的margin不会collapse
  • 一个in-flow block-level box的margin-bottom总是会与后面紧挨着它的in-flow block-level兄弟box的margin-top collapse,除非这个兄弟box有clearance
  • 一个in-flow block-level box的margin-top会与它的第一个in-flow block-level子box的margin-top collapse,条件是这个box没有border-top,没有padding-top,并且子box没有clearance
  • 一个height为auto且min-height为0的in-flow block-level box的margin-bottom会与它的最后一个in-flow block-level子box的margin-bottom collapse,条件是这个box没有bottom border,没有bottom padding,并且子box没有clearance

多个margin collapse以后形成的collapsed margin的width是这些被collapse的margin中的最大值.
如果2个margin中有一个正值和一个负值,collapse后的margin=正值-负值的绝对值
如果2个margin都是负值,collapse后的margin=0-较大的负值的绝对值

8.4.2 Border color

border-color属性指定了一个box中border area的color

border-top-color,border-right-color ,border-bottom-color,border-left-color
Value: color | transparent(边框是透明的) | inherit
Initial: the value of the 'color' property
Applies to: all elements
Inherited: no
Percentages: N/A(不接受百分比值)
Media:
Computed value: 如果从color取值就是color的computed value,否则就是specified value

如果一个元素的border-color没有指定,UA必须取该元素的color属性的值来作为border-color的computed value
