4.1.2.1 Vendor-specific extensions(特定供应商扩展)

Vendor-specific extensions应该遵循以下格式:
'-' + vendor identifier + '-' + meaningful name
'_' + vendor identifier + '-' + meaningful name

a:-webkit-any-link{...}
p{-moz-background-origin: border}

4.1.3 Characters and case

  1. 所有的CSS syntax是case-insensitive(大小写不敏感的),除了那些不由CSS控制的部分.例如,HTML属性'id'和'class'的值都是case-sensitive(大小写敏感的)
  2. CSS中,identifier(标识符)只能包含字母数字,以及连接符"-"和下划线"_",不能以数字开头
  3. 在CSS中,反斜线(\)字符可以表示3种类型的字符转义
    1. 在一个字符串内,后面跟着的换行的"\"会被忽略,看上去好像字符串既不含"\"也不含换行
    2. 取消特殊CSS字符的含义.例如,"\""
    3. 允许编写者引用那些难以在文档中直接输入的字符

4.1.7 Rule sets,declaration blocks,and selectors

body{font-size:16px;color:red;}

selector是"{"之前的部分.例如,body

declaration block,以"{"开始,"}"结束,之间必须由0或多个以";"分隔的declaration.例如,{font-size:16px;color:red;}

rule set由后面跟着一个declaration block的selector组成.例如,body{font-size:16px;color:red;}

4.1.8 Declarations and properties

body{font-size:16px;color:red;}

一个declaration由property name和":"和property value组成.例如,font-size:16px

4.1.9 Comments

注释以"/*"字符开始,以"*/"字符结束

body{font-size:16px;color:red;}     /*this is a comment*/

4.3 Values

4.3.1 Integers and real numbers

integer value用<integer>表示,由0-9的数字组成

real number value用<number>表示,一个<number>要么是一个<integer>,要么由数字后面跟符号"."后面跟数字组成

<integer>和<number>都只能用十进制数字

body{line-height:1.5;}       /*<number> value*/
.outer{z-index:1;counter-increment:2;}      /*<integer> value*/

4.3.2 Lengths

length涉及距离测量

length value用<length>表示,由一个<number>后面跟一个unit identifier(例如,px等)组成.值为0时后面的unit identifier是可选的

有2种length unit:relative和absolute.relative length unit在指定length时会相对于另一个length属性

relative length unit:

  1. em:相对于'font-size'
  2. ex:相对于'x-height'

'em'等于当前元素(即使用'em'的元素)'font-size'属性的computed value.例外情况是当在'font-size'属性值中使用'em'时,'em'等于父元素的'font-size'(详见'font-size')

'ex'由元素中的第一个可用font决定.'ex'等于当前元素中第一个可用font的'x-height'.例外情况是当在'font-size'属性值中使用'ex'时,'ex'等于父元素的'x-height'(详见'font-size')

h1{line-height:1.2em;}    /*h1的line-height比h1的font-size大20%*/
h1{font-size:1.2em;}    /*h1的font-size比h1继承的font-size大20%*/

当为document tree的root element指定relative length unit时,'ex'和'em'相对于CSS属性的initial value

子元素不会继承父元素的specified value(relative value),而是继承父元素的computed value(absolute value)

body{font-size:12px;text-indent:3em;}
h1{font-size:15px;}

如果h1是body的子元素,那么h1的text-indent是36px,而不是45px

absolute length unit彼此之间是固定不变的.包括物理单位(in,cm,mm,pt,pc)和像素单位(px)

  • in: 英寸 — 1in is equal to 2.54cm.
  • cm: centimeters
  • mm: millimeters
  • pt: points — the points used by CSS are equal to 1/72nd of 1in.
  • pc: picas — 1pc is equal to 12pt.
  • px: pixel units — 1px is equal to 0.75pt.

4.3.3 Percentages

percentage value用<percentage>表示,由一个<number>后面跟一个"%"组成

percentage value总是相对于另一个值.每一个接受percentage的属性也定义了这个percentage的参考值

4.3.4 URIs and URLs

URI value用<uri>表示,URI value在属性值中用url()表示

4.3.5 Counters

counter使用case-sensitive identifier表示(详见'counter-reset' and 'counter-increment')

要引用一个counter值的话,可以用标记'counter(<identifier>)'或者'counter(<identifier>,<'list-style-type'>)'

为了引用一系列嵌套的同名counter,可以用标记'counters(<identifier>, <string>)'或者'counters(<identifier>, <string>, <'list-style-type'>)'

有关counter的更多信息详见Automatic counters and numbering

4.3.7 Strings

字符串既能写在双引号里也能写在单引号里

字符串中不能直接包含newline(换行).为了在字符串中包含换行,可以用"\A"或者"\00000a",示例见'content'
