7.1 Introduction to media types

样式表最重要的特性之一是指定同一个文档在不同的media上如何显示

针对不同的media type的样式表可能共享一个属性,但是需要不同的属性值

7.2 Specifying media-dependent style sheets

目前有两种方式来指定样式表的media依赖:

  1. 在样式表内部通过@import or @media 2种at-rule来指定目标media的样式
  2. 在document language中指定目标media.例如在HTML中,可以通过link元素中的media属性给目标media指定一个样式表

7.2.1 The @media rule

@media rule指定一系列样式声明的目标media type(多个media type之间以","隔开).@media rule允许在一个样式表中针对不同media声明不同的样式

@media rule之外的样式声明适用于所有的media type

7.3 Recognized media types

all
适用于所有设备
print
用于在屏幕上用打印模式查看的文档
screen
用于计算机屏幕
speech
用于语音合成器
TV
用于电视类设备

media type名称是case-insensitive(大小写不敏感的)
