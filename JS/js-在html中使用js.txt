1、使用<script>元素的2种方式：
                           页面嵌入Javascript代码：<script type="text/javascript">
													function m1()
													{
														alert(hello,world);
													}
                                                   </script>
						            引入外部js文件：<script type="text/javascript"  src="../js/JavaScript1.js"></script> 与所在页面在同一个域
									                <script type="text/javascript"  src="http://www.baidu.com/js/JavaScript1.js"></script> 与所在页面不在同一个域
                                     外部js文件的好处：1、可维护性 2、可缓存，即浏览器可以根据具体的设置缓存链接所有的外部js文件


2、js代码执行顺序：每个<script>元素按在页面中出现的顺序从上到下依次被解释，在浏览器对<script>元素内部的代码解释完毕之前，页面中的其他内容不会被浏览器加载或显示
3、js在html中的位置：js引用放在<body>元素中页面内容的后面。这样在下载、解析和执行js之前页面的内容将全部被显示
4、<noscript>元素：这个元素用于在1>浏览器不支持js脚本2>浏览器js脚本被禁用的情况下显示
                   <noscript>本页面需要浏览器启用JavaScript</noscript>
5、浏览器在遇到<body>标签时才显示内容
