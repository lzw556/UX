关键字: let, const

let
作用域为当前所在块
同一个作用域内不能重复声明
不能提升

module

export
用于从模块中导出函数, 对象或原始值, 以便在其他模块中通过import语句使用它们
export语句不能用在嵌入式脚本中, 无论是否声明, 都处于严格模式
  • 命名导出(每个模块可包含任意数量)
    • eg. export let name1 = 5;
    • eg. export function methodName(){..};
    • eg. export {name1, methodName}
  • 默认导出(每个模块至多一个)
    • eg. export default function methodName(){...}
    • eg. export default {name1, methodName}
import
用于导入由另一个模块导出的绑定, 无论是否声明, 都处在严格模式
import语句属于静态导入. 在浏览器中, 只能在声明了 type="module" 的 script 的标签中使用

  • 导入模块的默认导出

    import 别名 from "modulePath"

  • 导入模块的命名导出

    import {命名1, 命名2, ...} from "modulePath"
