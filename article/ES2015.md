## let and const
* let(基本上修复了var的各种缺陷)
    * syntax, eg. let x = 1;  eg. let x = 1, y = 2;
    * 作用域为当前所在块
    * 没有声明提升, 所以必须先声明后使用
    * 同一个作用域内不能重复声明
* const
    * 声明时必须进行初始化
    * 保存基本类型值时, 是一个常量
    * 保存复杂类型值(比如对象)时, 是一个常量值指针(即变量指向不能修改, 但对象内部可以添加/删除/修改属性)

## 对象的增强
* 属性增强
    ```javascript
    let name2 = "lily", age2 =18;
    let obj2 = {name2, age2};
    console.log(obj2);
    ```
* 方法增强
    ```javascript
    let name4 = "lilei", age4 =18;
    let obj4 = { 
        name4,
        age4,
        read(){
            console.log(name4 + " reading");
        }
    }
    console.log(obj4);
    ```
