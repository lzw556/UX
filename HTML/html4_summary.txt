一 HTML基础

1 HTML标签和HTML元素 
   1.1 <h1>这个是一级标题</h1>,其中<h1></h1>是HTML标签,<h1>这个是一级标题</h1>是HTML元素;可以理解为HTML文档由HTML元素组成,HTML元素都是由HTML标签定义的
   1.2 HTML标签对大小写不敏感,但是小写最好
   1.3 没有内容的HTML元素叫做空元素,空元素在开始标签中关闭.比如:<br/>

2 HTML属性,HTML元素可以拥有属性
   2.0 小写
   2.1 以名/值对的形式出现,值始终要加双引号,除非这个值本身就包含双引号(这时才可以用单引号)
   2.2 在开始标签中定义
   2.3 适用于大多数HTML元素的属性:id class style title
   2.4 非常重要属性
          lang属性作用是定义文档语言,常用于html;
          title属性作用是当鼠标移到到所在元素上时title值会以提示条的形式显示;
          href属性作用是定义链接的地址,主要用于a;
          alt属性的作用是当一个html元素因为错误无法显示的时候浏览器会显示alt值,主要用于img;

3 HTML样式 
   3.1 外部样式表.当样式需要被应用到很多页面的时候,外部样式表是最好的选择.使用外部样式表,就可以通过改变一个文件来改变整个站点的外观.可以在head元素中通过<link>元素来定义
   3.2 内部样式表.当单个文件需要特别样式时使用内部样式表,可以在head元素中通过<style>标签来定义
   3.3 内联样式表.当特殊的样式需要应用到个别的元素时使用内联样式表,可以在元素中通过HTML标签的style属性来定义

4 HTML的注释语法 <!--...--> 

二 HTML元素

0 head,head元素中的内容不会被浏览器显示,主要包括描述html文档的元数据;常见的元数据元素有:
       title,title元素不会被浏览器显示在html文档中,而是显示在浏览器的tab页上;
       meta,常见的有:
             <meta charset="">
             <meta name="keywords" content="">
             <meta name="description" content="">
             <style></style>
             <link>

1 Headings,标题元素,由<h1> -<h6>等标签进行定义的
   1.1 共分6级, <h1>最大 <h6>最小
   1.2 <h1>会被搜索引擎抓到

2 Horizontal line,水平线,由<hr/>标签进行定义

3 paragraph,段落元素,由<p>标签进行定义;当浏览器在显示页面时会移除HTML源代码中多余的空格和空行,所有
  连续的空格或空行都会被算作一个空格.

4 line break,换行元素,由<br/>标签进行定义;一般用于在同一个段落内另起一个新行的情形

5 pre,用来定义预格式化的文本

6 html text formatting elements
    6.1 bold与strong: 2者的效果一致都是加粗文本;
         bold由<b>标签进行定义,表示加粗文本,没有加强的语义;
         strong由<strong>标签进行定义,表示加强文本,在语义上有强调的意思;
    6.2 italic与emphasize: 2者的效果一致都是斜体文本;
          italic由<i>标签进行定义,表示斜体文本,没有加强的语义;
          emphasize由<em>标签进行定义,表示加强文本,在语义上有强调的意思;
    6.3 small,小文本,由<small>标签进行定义;
    6.4 superscript,上标,由<sup>标签进行定义;
    6.5 subscript,下标,由<sub>标签进行定义;

7 超链接元素,由<a>标签进行定义
   7.1 有2种使用<a>标签的方式: 1>通过href属性指向另一个文档的链接 2>通过name属性创建文档内的书签
   7.2 herf属性,注意href的值后面始终要加上正斜杠/,假如这样书写链接：href="http://www.w3school.com.cn/html"，就会向服务器产生两次HTTP请求.这是因为服务器会添加正斜杠到这个地址，然后创建一个新的请求就像这样：href="http://www.w3school.com.cn/html/".
   7.3 target属性,定义被链接的文档在何处显示.当target="_blank"时会在新窗口中打开
   7.4 name属性,这个属性主要用于设置锚点.1>先创建锚点<a name="m1">锚点1</a> 2>创建指向该锚点的链接,<a href="#m1">这是一个链接,这个链接指向锚点1</a>
   7.5 图像链接:eg.<a href="xx"><img src="yy" alt=""/></a>
            
8 图像元素,由<img/>标签进行定义,其中alt属性表示当图片无法显示时,浏览器会显示alt属性的值;
   8.1 总是设置图片的宽和高,因为如果没有设置的话页面会在加载图片的时候闪动                                                           
9 表格元素,由<table>标签进行定义
   9.1 border属性,定义表格边框的宽度,最好用CSS来替代 eg.table,th,td{border:1px solid black;};
       border-collapse分2种:collapse和sperate(默认),最好用css来替代
        eg. table,th,td{
                 border:1px solid black;
                 border-collapse:collapse;
            }
   9.2 cellpadding属性,定义单元格边框与其内容之间的空白宽度,最好用CSS[padding]来替代
         eg. th,td{padding:10px;}
   9.3 cellspacing属性,定义单元格之间的空白宽度,最好用CSS[border-spacing]来替代
         eg. table{border-spacing:5px;}
   9.4 caption元素,由<caption>标签进行定义,定义表格的标题caption标签必须紧随table标签之后
   9.5 thead元素,由<thead>标签进行定义,定义表格的页眉
   9.6 tbody元素,由<tbody>标签进行定义,定义表格的主体
   9.7 tfoot元素,由<tfoot>标签进行定义,定义表格的页脚
   9.8 tr元素,由<tr>标签进行定义,定义表格的行
   9.9 th元素,由<th>标签进行定义,定义表格的表头 eg. th{text-align:center;}
   910 td元素,由<td>标签进行定义,定义表格的单元格
       1>colspan属性,规定单元格可以横跨的列数
       2>rowspan属性,规定单元格可以横跨的行数

10列表元素
   10.1 ordered lists,有序列表元素,由<ol>标签进行定义,每一个列表项由<li>标签进行定义;有默认的list-style
   10.2 unordered lists,无序列表元素,由<ul>标签进行定义,每一个列表项由<li>标签进行定义;有默认的list-style
   10.3 description lists,自定义列表元素,由<dl>标签进行定义,每一个列表项由<dt>标签和<dd>标签组成;其中dt用来表示该项的名称,dd用来表示该项的描述
   10.4 horizatal list,水平列表,用css中的li{display:inline;}来实现

11 容器元素
   11.1 div元素,没有特殊的含义,常见用途有2个:1>组合其他HTML元素的容器2>文档布局
   11.2 span元素,没有特殊的含义,常用来作为文本的容器

12表单,由<form>标签进行定义,是一个包含表单元素的区域,用于向服务器传输数据
   12.1 action属性,值是一个url,规定当表单提交时向何处发送表单数据 
   12.2 method属性,值:get/post,规定发送表单数据用那种http方法;
        什么时候用get:查询数据,提交信息中没有敏感信息
        用get有什么问题:表单数据会显示在URL中;数据量比较小
        什么时候用post:更新数据
   12.3 enctype属性,值:application/x-www-form-urlencoded  multipart/form-data text/plain,规定在发送表单数据之前如何对其进行编码
   12.4 fieldset,可以将表单中的相关元素分组,legend用来描述fieldset的标题

13表单元素,是允许用户在表单中（比如：文本域、下拉列表、单选框、复选框等等）输入信息的元素,一般放在表单中用于搜集用户输入
   13.1 输入元素,由<input>标签进行定义
         一 当type="text"/"password"时,可用属性有:
             1>name,用于对提交到服务器后的表单数据进行标识,必须设置,注意:只有设置了name属性的表单元素才能在提交表单时传递它们的值
             2>value,定义输入字段的初始值
             3>readonly,规定输入字段为只读
             4>disabled 规定应禁用输入字段,注意:被禁用的表单元素不会被提交,可有可无
         二 当type="radio"/"checkbox"时,可用属性有:
            1>name,必须设置,当type="radio"时,name属性的值相同的input元素是一组
            2>value,定义与输入想关联的值,必须设置
            3>checked,规定此input元素首次加载时应当被选中
            4>disabled 
         三 当type="hidden"时,可用属性有:
            1>name,必须设置 2>value,定义隐藏字段的值            
         四 当type="file"时,可用属性有:
            1>name,必须设置 2>disabled           
         五 当type="submit"/"reset"/"button"/"image"时,可用属性有:
            1>value 定义按钮上的显示的文本 2>disabled             
   13.2 下拉列表,由<select>和<option>2个标签组合起来进行定义    
         1>select元素的可用属性:name属性,必须设置; disabled                   
         2>option元素的可用属性:
           value属性,必须设置,定义每一个选项的值;selected属性,默认选中;disabled
   13.3 文本域,由<textarea>标签进行定义,可用属性有:
         1>name 2>readonly 3>disabled 4>cols,可见宽度 4>rows,可见高度
   13.4 总结:常见表单元素一共有:1(input[2(text,password)+2(radio,checkbox)+1(file)+1(hidden)+4(submit,reset,image,button)])+1(select)+1(textarea)
            
14框架集,由<frameset>和<frame>2个标签进行定义,用来在一个窗口中显示多个页面
   12.1 frameset的cols属性,规定每一列占据屏幕的宽度
   12.2 frameset的rows属性,规定每一行占据屏幕的高度
   12.3 frame的src属性,定义每一个框架中的显示的html文档
   12.4 frame的noresize属性,规定框架边框无法拖到
   12.5 注意:<frameset>和<body>不能共存 

15内联框架,由<iframe>标签进行定义,用于在网页中嵌入网页
   13.1 width属性,规定iframe的宽度
   13.2 height属性,规定iframe的高度
   13.3 frameborder,值:0/1,规定是否显示框架的边框
   13.4 scrolling,值:yes/no/auto,规定是否在iframe中显示滚动条     

三 HTML高级

1  HTML文档类型,由<!DOCTYPE>进行声明
   1.1 <!DOCTYPE>为浏览器提供一项声明,即HTML是用什么版本编写的,它不是HTML标签
   1.2 因为HTML有很多版本,所以只要完全明白页面中使用的HTML版本,浏览器才能完全正确地显示HTML页面  
   1.3 <!DOCTYPE>必须HTML文档的第一行,位于<HTML>标签之前
   1.4 常用的HTML文档类型:
       1>HTML5,<!DOCTYPE html>
       2>HTML4.0.1版本有3中文档类型:
         (1)HTML 4.01 Strict,<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">
         (2)HTML 4.01 Transitional,<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
         (3)HTML 4.01 Frameset,<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Frameset//EN" "http://www.w3.org/TR/html4/frameset.dtd">

2  HTML头部元素,由<head>标签进行定义,head是所有头部元素的容器.常见的头部元素有: 
   2.1 HTML标题元素,由<title>标签进行定义,title可以定义浏览器标题栏的标题;提供页面被添加到收藏夹时显示的默认标题
   2.2 <link>元素,定义HTML文档与外部资源之间的关系
       (1)常见的在HTML页面中引用外部CSS文件
       (2)网站左上角的图片,比如<link rel="shortcut icon" href="/favicon.ico" type="image/x-icon" />
   2.3 <style>元素,用于为HTML文档定义样式信息
   2.4 <script>元素,用于定义客户端脚本信息
   2.5 <meta>元素,提供关于HTML文档的元数据,元数据总是以名称/值的形式被成对传递的
       (1)name属性,定义元数据的描述(名称)
       (2)content属性,定义元数据的值
       (3)http-equiv属性,和name差不多,常见的元数据有:
          1> <meta name="keywords" content="HTML,CSS">,告诉搜索引擎页面的关键字
          2> <meta name="description" content="this is a website">,告诉搜索引擎页面的主要内容
          3> <meta name="robots",content="all/none/index">,告诉搜索机器人哪些页面需要索引
          4> <meta name="author",content="www.g.cn">,定义页面的作者
          5> <meta http-equiv="expries",content="Fri,12Jan200118:18:18GMT">,定义网页的过期时间,一旦过期,必须重新从服务器上获取.content的值必须用GMT的时间格式
          6> <meta http-equiv="pragma" content="no-cache">, 规定禁止浏览器从本地缓存中访问页面内容
          7> <meta http-equiv="refresh" content="5;url=http://www.g.cn">,自动刷新并跳转到指定页面,content中的5指停留5秒钟后跳转
          8> <meta http-equiv="content-type" content="text/html;charset=utf-8">,定义页面的字符编码
          9> <meta charset="utf-8"> HTML5的写法

3  HTML预留字符,HTML代码中不能使用预留字符,因为浏览器会误认为是标签,因此如果要显示预留字符,必须在HTML代码中用字符实体来代替预留字符,常见的预留字符和对应的字符实体有:
   显示    描述    字符实体名称  字符实体编号 
           空格      &nbsp;        &#160;
    <      小于号    &lt;          &#60;
    >      大于号    &gt;          &#62;
    "      双引号    &quot;        &#34;

四 XHTML,和HTML的区别有:
   元素语法:
     (1)XHTML元素必须正确嵌套
     (2)XHTML元素必须小写
     (3)XHTML元素必须关闭
   属性语法:
     (1)XHTML属性必须小写
     (2)XHTML属性值必须用引号包围
