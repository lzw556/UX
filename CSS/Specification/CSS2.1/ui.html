18.1 Cursors: the 'cursor' property

cursor
Value: [ [<uri>,]* [ auto | crosshair | default | pointer | move | e-resize | ne-resize | nw-resize | n-resize | se-resize | sw-resize | s-resize | w-resize | text | wait | help | progress ] ] | inherit
Initial: auto
Applies to: all elements
Inherited: yes
Percentages: N/A
Media:
Computed value: 除了把相对路径转换为绝对路径之外,和specified value一样

这个属性指定了指针设备上光标的显示类型

auto
由UA决定
crosshair
十字准线,瞄准线
default
依赖平台的默认光标.通常渲染为一个箭头
pointer
光标是一个指示链接的指针
move
指示某些东西可以被移动
e-resize, ne-resize, nw-resize, n-resize, se-resize, sw-resize, s-resize, w-resize
指示某些边界可以被移动
text
指示文本可以被选中
wait
指示程序正忙,用户应该等待.通常渲染为钟表或沙漏
progress
一个进度指示器.程序正在执行某些处理,但与'wait'的区别在于用户仍然可以与程序交互.通常渲染成一个旋转的沙滩球,或者一个带有钟表或沙漏的箭头
help
通过鼠标下方的对象可以获得帮助.通常渲染成一个问号或者气球
<uri>
UA从URI指定的资源中获取光标.如果UA无法处理光标列表中的第一个光标,它应该尝试处理第二个.如果UA无法处理所有用户定义的光标,它必须使用列表末尾的通用光标

18.4 Dynamic outlines: the 'outline' property

有时候,为了让可视化对象,例如按钮,活动表单字段等凸显出来,编写者可能想要在他们周围创建outline.CSS2.1中outline和border的区别在以下方面:

  1. outline不占据空间
  2. outline可能不是矩形
outline
Value: 'outline-style','outline-width','outline-color' | inherit
Initial: 见各个单独属性
Applies to: all elements
Inherited: no
Percentages: N/A
Media:
Computed value: 见各个单独属性
outline-style
Value: <border-style> | inherit
Initial: none
Applies to: all elements
Inherited: no
Percentages: N/A
Media:
Computed value: 和specified value一样
outline-width
Value: <border-width> | inherit
Initial: medium
Applies to: all elements
Inherited: no
Percentages: N/A
Media:
Computed value: 绝对长度,或者0如果'outline-style'值是'none'
outline-color
Value: <color> | invert | inherit
Initial: invert
Applies to: all elements
Inherited: no
Percentages: N/A
Media:
Computed value: 和specified value一样

outline总是位于一个box的上面,并且不会影响该box或其它box的位置与尺寸.因此,显示或者抑制outline不会导致reflow(重排)或者overflow(溢出)

outline从border边界外开始渲染

outline可以不是矩形的.例如,如果元素被拆分到了多行,outline是能把元素的所有盒包住的最小outline.与border相比,outline在line box的开始处或者结束处不是断开的,而是总会尽量完全连接起来

'outline-style'和'border-style'接受相同的值,除了'hidden'不是合法的outline style

'outline-width'和'border-width'接受相同的值

'outline-color'接受所有的color值和关键字'invert'.'invert'用来对屏幕上的像素进行反色.这是一种用来确保焦点边框可见的常见技巧,无论背景色是什么

所有方向上的outline是一样的.与border相比,不存在'outline-top'或'outline-left'
