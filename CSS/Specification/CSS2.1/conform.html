3.1 Definitions

Source document
应用了一个或多个样式表的文档.以某种语言编码,并用元素树表示文档
Document language
source document的编码语言(例如,HTML,XHTML).CSS用来描述document language的表现,但不会改变document language的基本语义
Replaced element
内容超出CSS formatting model的元素.例如,img.Replaced element通常拥有intrinsic dimensions(固有尺寸):intrinsic width,intrinsic height和intrinsic ratio
Intrinsic dimension
由元素自身定义的宽度和高度,不受环境影响.在CSS2.1中,只有Replaced element拥有intrinsic dimensions
Content
source document中元素相关的内容,一些元素没有内容,被称为empty element
Rendered content
根据相关样式表对元素进行渲染后元素的内容
Document tree
source document中的元素树.树上的每一个元素都有且只有一个父元素,除了根元素
Child
元素A叫作元素B的子元素当且仅当元素B是元素A的父元素
Descendant
包括子,孙,...,子子孙孙
Ancestor
元素A叫作元素B的祖先元素当且仅当元素B是元素A的后代元素
Sibling
元素A叫作元素B的兄弟元素当且仅当元素B和元素A share同一个父元素.如果文档树中A在B之前,A就是前导兄弟.如果文档树中B在A后面,B就是后继兄弟
