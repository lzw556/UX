@media all and (max-width:800px){
    p#id1{color:navy;}
}

这个P元素,id属性值是'id1',当viewport宽度(UA的窗口宽度)被缩放至小于等于800px时,其文本颜色应该是navy

@media (max-width:800px){
    h3{font-weight:normal;}
}

这个H3元素,当viewport宽度(UA的窗口宽度)被缩放至小于等于800px时,其字体会由粗体变成正常

@media print{
    p#id2{font-size:30px;}
}

这个P元素,id属性值是'id2',当用打印模式查看本文档时,其字体大小会变成30px

@media (orientation:portrait){
    p#id3{background-color:navy;
   color:white;}
}

这个P元素,id属性值是'id3',当viewport是竖屏(高度大于宽度)时,其背景颜色应该是navy,文本颜色应该是white
