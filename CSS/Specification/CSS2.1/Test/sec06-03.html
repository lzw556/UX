@import url("styleimport.css");   /* @import规则必须写在最前面,否则无效 */
@import "styleimport/styleimport.css";
.cl1{
    font-weight:bold;
    color:navy;
    background-color:silver;
}

下面每一种情况应用的样式声明都是: {font-weight:bold; color:navy; background-color:silver;}

这个P元素,字体应该是粗体,文本颜色应该是navy,背景颜色应该是silver,这些样式声明来自于行内样式表(style属性)

这个P元素,class属性值是'cl1',字体应该是粗体,文本颜色应该是navy,背景颜色应该是silver,这些样式声明来自于嵌入样式表(style元素)

这个P元素,class属性值是'styleimport',字体应该是粗体,文本颜色应该是navy,背景颜色应该是silver,这些样式声明是通过嵌入样式表(style元素)中的@import导入的
