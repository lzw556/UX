内联非替换元素有2个height:content-area和virtual-area(line-height)
  1. content-area
    1. 在本例中padding:0,所以content-area的区域就是绿色背景区域
    2. 由font-family决定,指定font-size:100px下,content-area的height分别是:
      microsoft yahei:132px
      arial:111px
      Times New Roman:112px
  2. virtual-area
    1. 就是line-height,即inline-box的height
    2. 在本例中line-height:normal,其中normal的值由font-family决定
      指定font-size:100px下,virtual-area的height分别是:
      microsoft yahei:132px,即normal:1.32
      arial:115px,即normal:1.15
      Times New Roman:115px,即normal:1.15
